## Run Locally

**Prerequisites:**  Node.js


1. Download the files in this repository (Code -> Download Zip)
2. Copy the unzipped files into a location on your C drive. Make a note of this path.
3. Go to cmd
4. Change directory to the path above
5. Type in: npm install
7. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
8. Type in: npm run dev
9. You'll find a screen that takes you to a local host browser window
10. You can then run the game locally!
